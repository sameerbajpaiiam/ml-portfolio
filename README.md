Interactive web application that showcases **two production-ready end-to-end ML projects** with real-time inference.

## Projects Included
### 1. Predictive Environmental Modelling & Multivariate Air Quality Analysis
- Multi-output regression (Temperature, Relative Humidity, Absolute Humidity)
- Models: **XGBoost** + **Random Forest**
- Achieved **R² = 0.91** | **RMSE = 4.10**
- Feature importance comparison identifying CO, NO₂, and NMHC as top predictors

### 2. Industrial Casting Defect Detection
- Binary image classification (defective vs non-defective casting products)
- Models: **Custom CNN** + **MobileNetV2** (transfer learning)
- Rigorous evaluation with classification reports, confusion matrices, loss/accuracy curves

## Features
- Clean, responsive Streamlit interface with sidebar navigation
- Dynamic input forms and on-the-fly predictions
- Image upload + instant computer-vision diagnostics
- Fully deployed for public access and usability testing
