An interactive web application built with **Streamlit** that showcases end-to-end Machine Learning workflows with real-time model predictions.

## Features
- Dynamic input handling and on-the-fly inference for two production-ready ML models
- Integrated **Casting Defect Detection** (CNN + MobileNetV2)
- Integrated **Multivariate Air Quality Prediction** (XGBoost regression)
- Clean, user-friendly interface for public access and usability testing
- Fully deployed and accessible online

## Tech Stack
- **Frontend**: Streamlit
- **Backend**: Python, Scikit-learn, TensorFlow, XGBoost
- **Models**: MobileNetV2 (transfer learning), Random Forest, XGBoost
- **Deployment**: Streamlit Community Cloud

## Project Background
Designed and deployed as part of my Machine Learning portfolio to demonstrate practical ML deployment capabilities, including data preprocessing, feature engineering, and model pipelines.
